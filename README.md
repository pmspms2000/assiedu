# AssiEdu

교환학생 영어 수업용 실시간 번역·설명 데스크톱 AI (macOS).
교수님 말씀을 마이크로 듣고 → 영어 원문 + 한국어 번역을 실시간 자막으로 보여주며,
필요할 때 용어 설명과 수업 요약을 만들어 줍니다.

## 구조

- **Electron 데스크톱 앱** — 수업 화면 위에 떠 있는 자막 창
- **음성 → 텍스트**: Deepgram 실시간 스트리밍 (영어)
- **번역 / 설명 / 요약**: Claude API
  - 실시간 번역: `claude-haiku-4-5` (저지연)
  - 용어 설명 · 수업 요약: `claude-opus-4-8` (고품질)
  - 모델은 `src/claude.js` 상단 상수에서 변경 가능

## 준비

1. **API 키 발급**
   - Claude: https://console.anthropic.com
   - Deepgram: https://console.deepgram.com (무료 크레딧 제공)
2. `.env.example` 을 `.env` 로 복사하고 두 키를 채웁니다.

```
cp .env.example .env
```

3. 의존성 설치 및 실행

```
npm install
npm start
```

## 사용

1. `▶ 시작` 을 누르면 마이크 권한을 요청합니다 (최초 1회).
2. 교수님 말씀이 영어 원문 + 한국어 번역 자막으로 누적됩니다.
3. 각 자막의 `💡 설명` 버튼으로 그 부분의 용어·개념 풀이를 볼 수 있습니다.
4. 수업이 끝나면 `📝 수업 요약` 으로 전체 복습 노트를 생성합니다.
   요약본은 `~/Documents/AssiEdu/` 에 마크다운으로 저장됩니다.

## 입력 장치 선택

상단 `🎤 입력` 드롭다운에서 소리를 받아올 장치를 고릅니다. `🔄` 로 목록을 새로고침.

- **대면 수업**: 노트북 **내장 마이크** 선택 (교수님 목소리를 마이크로 들음)
- **이어폰/외장 마이크**: 연결 후 `🔄` → 목록에서 선택
- **녹화 강의 / 온라인 영상의 소리**: 영상에서 나오는 **시스템 오디오**를 직접 인식하려면
  [BlackHole](https://github.com/ExistentialAudio/BlackHole) (무료 가상 오디오 장치)을 설치하세요.
  - 설치: `brew install blackhole-2ch` (또는 공식 설치파일)
  - macOS `Audio MIDI 설정`에서 **다중 출력 장치**(스피커 + BlackHole)를 만들어 시스템 출력으로 지정하면,
    영상을 들으면서 동시에 그 소리가 BlackHole로도 흘러갑니다.
  - 그 다음 이 앱의 입력 드롭다운에서 **BlackHole 2ch** 를 선택하면 강의 소리를 깨끗하게 인식합니다.

> Mac mini 등 마이크가 없는 데스크톱은 입력 장치가 없으니, 이어폰/마이크를 연결하거나 BlackHole을 쓰세요.

## 참고

- API 키는 메인 프로세스에서만 다루며, 자막 창(렌더러)에 직접 노출되지 않습니다.
