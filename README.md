**영어 강의가 버거운 학생을 위한 실시간 강의 도우미.**
교수님이 영어로 말해도, 화면에 **원문 + 한국어 번역 자막**이 같이 뜨고,
모르는 용어는 **설명**해주고, 궁금한 건 **질문**하고, 수업이 끝나면 **요약 노트**까지 만들어줍니다.

> 교환학생·영어 전공수업·플립러닝 등 *"영어라서 따라가기 힘든"* 모든 상황을 위해 만들었습니다.
> **온라인(녹화강의·줌·유튜브)과 오프라인(강의실 대면수업) 둘 다** 커버합니다.

데스크톱 앱(맥·윈도우)으로, 강의 화면 위에 **항상 떠 있는 자막 창**으로 동작합니다.

## 이 앱이 해결하는 문제

영어 강의를 들을 때 흔한 어려움:

- 교수님 말이 **빠르고 전문용어가 많아** 핵심을 놓친다
- 받아쓰기·번역을 직접 하다 보면 **수업 흐름을 따라가지 못한다**
- 슬라이드/판서를 봐도 **무슨 개념인지 바로 이해가 안 된다**
- 복습하려 해도 **뭐가 중요했는지** 정리가 안 된다

AssiEdu는 이 과정을 자동화해서, 학생이 **이해와 집중에만** 신경 쓰게 합니다.

## 주요 기능

| 기능 | 설명 |
|---|---|
| 🗣️ **실시간 자막** | 영어 강의를 들으며 **영어 원문 + 한국어 번역**을 자막으로 동시 표시 |
| 🇰🇷 **한국어 받아쓰기** | 한국어 강의도 받아써서 보여줌 — 플립러닝/집중이 흐트러질 때 따라 읽기용 |
| 🖥️ **화면·판서 보기** | 지금 보는 슬라이드/판서를 AI가 함께 "보고" 더 정확히 설명·답변 |
| 💡 **용어 설명** | 자막의 어려운 개념을 버튼 한 번으로 한국어로 풀어서 설명 |
| 💬 **질문 챗봇** | 별도 창에서 질문 → **강의 자막 + 화면/판서 + 이전 대화**를 근거로 한국어 답변. 창을 닫았다 열어도 대화 유지 |
| 📝 **수업 요약** | 지금까지의 강의를 **한국어 복습 노트(마크다운)**로 정리해 저장 |

## 온라인 / 오프라인 모두 지원

소리를 어디서 받느냐만 고르면 됩니다 (앱의 **🎤 입력** 메뉴).

- **오프라인(강의실 대면수업)** — 노트북 **내장 마이크**, 또는 연결한 **외장 마이크/이어폰**으로 교수님 목소리를 인식
- **온라인(녹화강의·줌·유튜브 등)** — **🎬 화면·영상 소리** 를 선택하면 컴퓨터에서 재생되는 **시스템 오디오**를 그대로 인식 (스피커로 다시 들려줄 필요 없음)

강의 언어도 **🗣️ 강의** 메뉴에서 고릅니다:
- 🇺🇸 **영어 강의 → 한국어 번역**
- 🇰🇷 **한국어 강의 → 받아쓰기**

## AI 제공자 — 무료부터 내 계정까지

**음성 인식(자막)은 항상 무료**로 동작합니다(로컬 Whisper, 키 불필요).
**번역·설명·질문·요약**에 쓸 AI는 ⚙️ 설정에서 고릅니다:

| 선택 | 필요한 것 | 비고 |
|---|---|---|
| **무료** | 없음 | 번역=MyMemory. 설명·요약·질문은 꺼짐. 가장 간단 |
| **Claude (Anthropic API 키)** | 본인 API 키 | 고품질 + 화면/판서 읽기 지원 |
| **GPT (OpenAI API 키)** | 본인 API 키 | 고품질 + 화면/판서 읽기 지원 |
| **내 컴퓨터의 Claude** | [Claude Code](https://claude.com/claude-code) 로그인 | 키 불필요(내 계정). 화면/판서 읽기 지원 |
| **내 컴퓨터의 Codex** | Codex CLI `codex login`(ChatGPT 계정) | 키 불필요(내 계정). **텍스트 전용** — 화면/판서는 못 봄 |

> - API 키는 **이 기기에 OS 키체인으로 암호화 저장**되며, 외부로 전송되지 않습니다.
> - "내 컴퓨터의 Claude/Codex" 옵션은 해당 CLI가 설치·로그인돼 있을 때만 설정에 나타납니다.
> - 비용은 각자 본인 계정으로 청구됩니다(무료 모드는 비용 없음).

## 설치

### 맥
1. `AssiEdu-<버전>-arm64.dmg` 더블클릭 → **AssiEdu**를 응용 프로그램 폴더로 드래그
2. 처음 실행 시 서명이 없어 막히면: 앱 **우클릭 → 열기 → 열기**
3. **마이크**, **화면 기록** 권한을 허용 (녹화강의 소리 인식에 화면 기록 권한 필요)

### 윈도우
1. `AssiEdu Setup <버전>.exe` 실행 → 설치 (바탕화면/시작메뉴 바로가기 생성)
2. SmartScreen 경고 시 **추가 정보 → 실행**
3. **마이크** 권한 허용

> 직접 빌드하거나 두 OS 설치파일을 한 번에 만드는 방법은 **[BUILD.md](BUILD.md)** 참고.

## 사용법

1. **🗣️ 강의** 에서 강의 언어 선택 (영어→번역 / 한국어→받아쓰기)
2. **🎤 입력** 에서 소리 출처 선택 (강의실=마이크 / 녹화강의=화면·영상 소리)
3. (선택) **⚙️ 설정**에서 AI 제공자 선택 — 설명·질문·요약을 쓰려면 무료가 아닌 옵션 선택
4. **▶ 시작** → 자막이 뜨기 시작
5. (선택) **🖥️ 화면 보기**를 켜면 슬라이드/판서까지 AI가 참고
6. 궁금하면 **💬 질문** 으로 물어보고, 수업 끝나면 **📝 요약**

> 자막 창과 질문 창은 **따로 떠서 나란히** 놓을 수 있고, 둘 다 다른 창 위에 항상 표시됩니다.

## 개인정보 / 처리 위치

- **음성 인식**은 인터넷 없이 **기기 안에서**(로컬 Whisper) 처리됩니다.
- **번역·설명·질문·요약**은 고른 제공자로 전송됩니다(무료=MyMemory, 키/계정=해당 AI). 무료 모드 외에는 해당 서비스로 텍스트/이미지가 전송됨에 유의하세요.
- API 키는 기기에 암호화 저장되며 코드 저장소에는 포함되지 않습니다.

## 기술 메모

- **Electron** 데스크톱 앱(맥/윈도우 공용), 항상-위 자막 창
- **음성 인식**: `@huggingface/transformers`(Whisper) — WebGPU/WASM, 무음 기반 청크 분할
- **무료 번역**: MyMemory 공개 API
- **AI 제공자 라우팅**(`src/ai.js`): Anthropic SDK / OpenAI API / 로컬 `claude`·`codex` CLI
- **화면/판서**: `getDisplayMedia`로 프레임 캡처 → 멀티모달 모델에 이미지로 전달
- 설정/키 저장: Electron `safeStorage`(OS 키체인)

자세한 빌드·배포는 **[BUILD.md](BUILD.md)** 참고.
